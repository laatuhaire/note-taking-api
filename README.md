# **TITLE:** NOTE TAKING API
# **Description:** A simple  API that allows users to do the followingcreating, reading, updating, and deleting notes using Node.js and Express. Data is stored in memory, making it ideal for learning and small projects
# Features
1)Create a new note
2)Retrieve all notes
3)Retrieve a single note by ID
4)Update a note by ID
5)Delete a note by ID
